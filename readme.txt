== Description ==

= WordPress NeePic Plugin =

<a href="http://neepic.net/" target="_blank">NeePic</a> - Cloud Image Hosting. Benefits of storing images with NeePic:

* Unlimited storage time. Period of storage is unlimited for the images, which were uploaded via plugin
* 99.99% guarantee of safety. By using the cloud technology, we can surely guarantee of 99.99% safety for all images
* Free. We take care of all costs
* Faster loading. The loading speed increases due to the load distribution inside the cloud
* Statistics. Each image has a counter of views
* Multilingual

== Installation ==

1. Download and unzip plugin files into folder <code>wp-content/plugins/</code>
3. Activate plugin in admin panel
4. Done
